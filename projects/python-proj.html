Lista de Atividades

Sobre o Projeto

Projeto para uso pessoal. Visa-se criar uma lista de atividades que possa organizar melhor o tempo que vou tomar em cada atividade e posteriormente o descanso.

A inserção das atividades devem ter o padrão ReGEX de "copia e cola" das atividades da platafroma "professorferretto.com.br" na versão de 2024.

Além desse padrão da plataforma Ferretto, deve-se aceitar a inserção manual por meio de outro pattern estabelecido, para inserção de atividades domésticas e tempo de descanso.

Neste projeto utilizei as bibliotecas:

  • PyQt6 (Para interface gráfica)
  • sqlite3

Cronograma Flexível

Desenvolvido perto das férias de meio de ano do 2⁰ ano do ensino médio para organizar meus estudos. Equilibrando as aulas com atividades, simulados e aulas de revisões.

Agora vem a dúvida: Faz sentido fazer um app somente para isso? SIM! A minha rotina basicamente era: ir para escola de período integral, voltar para casa, fazer exercício físico, comer e estudar em casa . Após os estudos, eu fazia uma lista do que teria de estudar no dia seguinte, encaixando perfeitamente cada aula, exercício... tempo de descanso para formar blocos de estudo (caso não saiba o que é, este vídeo irá ajudar: blocos de estudo)

Como muito frequentemente ocorriam imprevistos, aquele planejamento que eu tinha feito no dia anterior era basicamente perdido (E GASTA UM BOM TEMPO PARA CRIAR UM NOVO). Além disso, conforme ia avançando nas matérias, ficava cada vez mais complicado de fazer os exerícios de outros asssuntos da mesma matéria(a famosa revisão espaçada), com isso, uma carga de tempo cada vez maior era criada para cada planejamento.

Com este grande problema em mãos, resolvi criar o app. Demorou cerca de um 4 semanas para fazer (tive que aprender a lib PyQt6 que é relativamente complexa, e possui uma documentação um tanto confusa).

Usei ele por um bom tempo, e realmente resolveu o problema descrito.

Neste projeto utilizei as bibliotecas:

  • PyQt6 (GUI)
  • qt_material
